This documentation is for INTERNAL USE ONLY and MUST NEVER BE MADE PART OF A RELEASE of the software.

To update the documentation (once you've edited some Markdown files) change to this directory and then run ./generate_docs.sh
To view the generated documentation launch the index.html file from the "site" directory contained within this directory.
If you want to just view the live docs before generating an update then change to this directory and then run ./serve_live_docs.sh
